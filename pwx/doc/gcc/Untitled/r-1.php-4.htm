Шаг 4 - Начало установки

Чтобы начать установку нам надо вызвать программу setup: 

# setup

После этого запустится меню установочной программы. 

HELP       Read the Slackware Setup HELP file
KEYMAP     Remap your keyboard if you're not using a US one
ADDSWAP    Set up your swap partition(s)
TARGET     Set up your target partitions
SOURCE     Select source media
INSTALL    Install selected software
CONFIGURE  Reconfigure your Linux system
EXIT       Exit Slackware Linux Setup

Думаю тут все ясно. Установку надо начинать с второй позиции KEYMAP, но мы ее пропустим, потому что потом сможем поменять все внутри системы. И вообще на этапе установки проще делать меньше всего настроек, все равно потом все лучше менять самому.

Перейдем к разделу добавления своп-раздела, в меню ADDSWAP и нажмем Enter. Появится окно я перечислением разеделов диска типа Linux swap: 

Slackware Setup has detected a swap partition:

   Device Boot    Start       End    Blocks   Id  System
/dev/hda1             1         9     72261   82  Linux swap


Do you wish to install this as your swap partition?

Конечно хотим, выбираем кнопку < Yes >. После этого раздел быстро отформатируется и подключится. Он также добавится в файл /etc/fstab: 

Your swapspace has been configured. This information will
be added to your /etc/fstab:

/dev/hda1       swap      swap     defaults   0   0

Нажимаем кнопку < EXIT > и попадаем в следующий раздел главного меню - выбор диска для установки: 

Now that you've set up your swap space, you may continue
on with the installation. Otherwise, you'll be returned
to the main menu. Would you like to continue the 
installation and set up your TARGET drive(s)?
         < Yes >       < No >

Мы продолжаем не выходя в главное меню, поэтому выбираем кнопку < Yes >. После этого программа установки выведет все найденные ею разделы пригодные для установки. 

Please select a partition from the following list to use for your
root (/) Linux partition.

/dev/hda2  Linux native 4120627K

Выбираем этот самый /dev/hda2 и нажимаем < OK >. После этого выведется окно с предложением отформатировать данный раздел. 

Format  Quick format with no bad block checking
Check   Slow format that checks for bad blocks
No      No, do not format this partition
         < OK >       < Cancel >

У нас раздел новый и не отформатированный, поэтому его надо обязательно отформатировать. Чтобы сэкономить время можно выбрать Format, а если Вы хотите еще проверить диск на сбойные блоки, то выбирайте Check. Если Вы ставите систему на готовый заполненный чем-то раздел (например переустанавливаете), то лучше Вам его не трогать, и выбрать No. Я не буду терять время и выберу простое форматирование Format.

После того, как пройдет процесс форматирования можно будет приступать к следующей стадии установки.
